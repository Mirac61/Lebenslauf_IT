# Interaktiver Lebenslauf

Ein moderner, interaktiver Lebenslauf gebaut mit HTML, CSS und JavaScript.  
Fokus: sauberes Developer(Apple)-Design, klare Struktur und ein ruhiges UI.

## Features
- Light/Dark-Mode  
- Dynamische Sektionen (ein-/ausblendbar)  
- Responsive Layout  
- Fokus auf Lesbarkeit und Struktur

## Tech Stack
HTML · CSS · JavaScript
